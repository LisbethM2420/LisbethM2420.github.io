Detalles del Curso

Descripción

Este curso te enseñará a utilizar Microsoft Word desde lo más básico hasta un nivel intermedio, permitiéndote crear documentos profesionales con facilidad y eficiencia.

Objetivos de Aprendizaje
  • Familiarizarse con la interfaz de Microsoft Word
  • Dominar las funciones básicas de formato de texto
  • Crear documentos con estructura profesional
  • Utilizar tablas y gráficos simples
  • Revisar y corregir documentos eficientemente
Requisitos

No se requieren conocimientos previos. Solo necesitarás una computadora con Microsoft Word instalado (cualquier versión reciente).
